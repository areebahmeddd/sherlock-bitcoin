## Heuristic Catalogue

| Heuristic | Description |
|-----------|-------------|
| `cioh` | Common Input Ownership — multiple inputs likely from same wallet |
| `change_detection` | Identifies likely change output via script type matching, round numbers, and smallest output |
| `coinjoin` | Detects equal-value outputs with many inputs (CoinJoin pattern) |
| `consolidation` | Many inputs merged into 1–2 outputs of the same type |
| `self_transfer` | All outputs match input script type — likely internal wallet move |
| `address_reuse` | Same address appears in both inputs and outputs |
| `op_return` | OP_RETURN output detected, protocol identified |
| `round_number_payment` | Output value is a round BTC/sat amount |
| `peeling_chain` | 1 input → 1 small + 1 large output (peeling chain pattern) |
